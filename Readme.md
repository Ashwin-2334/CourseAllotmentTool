# Course Allotment Tool for Teachers

This is a web-based application developed using Flask(Python) and web based front-end to automate the process of course allotment for faculty members.
The system assigns courses based on predefined criteria and supports multiple input formats such as Excel and Word files.
It can also generate detailed reports in both formats, reducing manual effort and improving efficiency.

## Features

- Automatic course allotment based on predefined rules.
- Supports Excel (.xlsx) and Word (.docx) input files.
- Generates reports in Excel and Word formats.
- Simple and responsive HTML interface.
- Error handling for smooth backend processing.

## Tech Stack Used

- Backend: Flask(Python)
- Frontend: HTML, CSS
- Libraries Used: pandas, openpyxl, python-docx
